# Propridict

ProPriDict is a simple machine-learning based project that predicts current and future land or property prices.
By using features like location, area, number of rooms, and property details, the model provides an estimated price and also predicts how the price may increase in the coming years.

This project helps users make better buying, selling, and investment decisions using data-driven predictions.
